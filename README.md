# Granite Field Holdings, Ltd. Co. — Corporate Website

Official corporate website for **Granite Field Holdings, Ltd. Co.**, a private holding company engaged in strategic asset and intellectual property management, registered in the State of New Mexico, USA.

🌐 **Live Site:** [granitefieldholdings.com](https://granitefieldholdings.com)

---

## About

This repository contains the source code for the official web presence of Granite Field Holdings, Ltd. Co. The site is a fully static, single-file HTML website requiring no build tools, frameworks, or backend infrastructure.

---

## Pages

- **Overview** — Corporate identity and entity classification
- **About** — Company background, operational model, and core principles
- **Governance** — Operational areas and structural mandate
- **Compliance & Legal** — Regulatory disclosures and legal notices
- **Contact** — Corporate contact details and engagement policy

---

## Tech Stack

- HTML5 / CSS3 / Vanilla JavaScript
- No frameworks, no dependencies, no build step
- Google Fonts: `Cormorant Garamond` + `Jost`
- Fully responsive — mobile and desktop
- ARIA-compliant semantic markup

---

## Deployment

### GitHub Pages

1. Clone the repository
2. Rename `granite-field-holdings.html` to `index.html`
3. Go to **Settings → Pages** and set the source branch
4. The site will be live at `https://<username>.github.io/<repo>/`

### Netlify

Connect this repository to Netlify with:
- **Build command:** *(none)*
- **Publish directory:** `.`

### Vercel

```bash
npx vercel --prod
```

### Custom Domain

| Record | Name | Value |
|---|---|---|
| `A` | `@` | Your host IP |
| `CNAME` | `www` | Your Netlify / Pages / Vercel domain |
